# Product Overview

A Next.js web application with modern UI components and animations. The project combines shadcn/ui for accessible, customizable components with Aceternity UI for advanced animated effects and visual elements.

## Key Features

- Modern, responsive web interface
- Dark mode support with CSS variables
- Animated UI components with Framer Motion
- Accessible component library (shadcn/ui)
- Custom font integration (Geist Sans & Geist Mono)
